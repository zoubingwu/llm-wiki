# Web Audio API

Web Audio API 是浏览器中用于音频解码、播放、分析与处理的一组接口。

## 在文章中的用途

这篇文章使用 Web Audio API 来：

- 读取音频文件
- 解码出 `AudioBuffer`
- 获取单个声道的样本数组
- 根据当前播放时间确定分析窗口

## 对音频可视化的重要性

在 Web 环境下，Web Audio API 是连接“音频播放”和“频谱计算”的基础设施。
