# qmd

qmd 是一个面向 Markdown 文件的本地搜索工具，支持混合 BM25 / 向量搜索以及 LLM 重排序。

## 在当前 wiki 里的相关性

在 [[Retrieval-Augmented Generation (RAG)]] 和 [[LLM Wiki Pattern]] 的语境下，qmd 被视为随着 wiki 规模增长后可引入的搜索基础设施。

## 适用场景

- 对大量 Markdown 页面做本地搜索
- 为个人 wiki 提供比 `index.md` 更强的检索能力
- 作为 LLM 可调用的 CLI 或 MCP 工具
