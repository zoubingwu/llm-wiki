# Free Software Movement

自由软件运动（Free Software Movement）是由 Richard Stallman 和 GNU 传统代表的一套软件自由理念与运动，强调用户运行、研究、修改和分享软件的自由。

在 [[How To Become A Hacker]] 的历史说明中，[[Eric Steven Raymond]] 把自由软件运动描述为对早期黑客共同体中源码共享实践的命名与意识形态化。文章认为，很多黑客接受共享源码和协作开发实践，同时对“free software”标签的政治含义有不同程度的保留。

## 与 Open Source 的分化

[[Open Source Software]] 这个术语在 1998 年出现后，被许多黑客和 Linux 社区成员快速采用。文章把 open source 描述为更能容纳多样化开发者和项目的标签，并记录了它与自由软件运动之间的历史张力。
