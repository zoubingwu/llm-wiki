# Chroma

Chroma 是一家人工智能公司，发布了关于向量数据库和 LLM 的研究。

## 相关研究

- [Context Rot: How Increasing Input Tokens Impacts LLM Performance](https://research.trychroma.com/context-rot)（2025）
  - 测试了 18 个前沿模型（GPT-4.1、Claude、Gemini 等）
  - 所有模型都存在随着输入增长性能下降的问题
  - 下降非渐进式，而是存在临界点后断崖式下降
