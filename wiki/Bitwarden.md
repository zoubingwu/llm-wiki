# Bitwarden

Bitwarden 是一个密码管理器产品和公司，提供云托管密码库、企业功能、自托管服务端，以及开源客户端。[[The Quiet Renovation at Bitwarden - ByteHaven - Where I ramble about bytes]] 把它作为开源产品信任变化的案例。

## 文章中的变化

- Premium 价格在 2026 年前后翻倍，价格变化被放进功能公告中说明。
- 长期 CEO Michael Crandell 转为顾问角色，Michael Sullivan 接任 CEO。
- Sullivan 的公开履历强调并购（M&A）、私募股权（Private Equity）整合和退出经验。
- “Always free” 作为免费层承诺的页面文案曾经消失，文章传播后又重新出现在价格表中。
- GRIT 价值观中的 Inclusion 和 Transparency 被 Innovation 和 Trust 替换。

## 与 Vaultwarden 的关系

[[Vaultwarden]] 通过实现 Bitwarden server API，让用户可以自托管兼容服务端，并继续使用官方 Bitwarden 客户端。

这让 Bitwarden 的客户端许可证、API 稳定性和第三方服务器连接策略变成自托管用户的关键风险面。文章把这个风险压缩成一个观察点：关注客户端是否继续开源，以及 API 是否出现让第三方服务器难以跟进的漂移。
