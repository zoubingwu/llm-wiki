# Framer Motion

Framer Motion 是一个用于 React 的动画库，以声明式 API 和较好的交互手感著称。

## 在当前 wiki 里的相关性

这篇文章以 Framer Motion 的弹簧动画为例，解释了前端动画中的物理模型如何工作。

## 关键点

- 支持基于 [[Spring Animation]] 的动画
- 常见可调参数包括 [[Spring Parameters in Framer Motion|mass、stiffness、damping]]
- 适合实现更自然的 UI 反馈和过渡效果
