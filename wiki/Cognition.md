# Cognition

Cognition 是 Devin 编码 agent 的开发商。

## 相关观点

- 认为具有良好压缩能力的**单 agent** 比多 agent 架构提供更稳定的结果和更低的成本[[Multi-agent System]]
- 专门训练了一个独立的模型用于 agent 之间的摘要/压缩，说明不良压缩的严重后果[[Context Engineering]]
- 见 [Context Engineering for Agents](https://blog.langchain.com/context-engineering-for-agents/)
